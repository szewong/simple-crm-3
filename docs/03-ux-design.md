## Design Philosophy

Modern, clean SaaS application inspired by Linear, Notion, and Attio. The interface prioritizes clarity, speed, and minimal cognitive load. Every screen follows these principles:

- **Minimal but powerful** â€” hide complexity until needed
- **Spacious layouts** â€” generous whitespace, clear visual hierarchy
- **Excellent typography** â€” Inter font family, well-defined type scale
- **Subtle animations** â€” smooth transitions (150-300ms), micro-interactions on hover/focus
- **Consistency** â€” shared patterns across all entity types (contacts, companies, deals)

### Color System
- **Primary**: Indigo-600 (`#4F46E5`) â€” buttons, links, active states
- **Neutral**: Slate palette â€” text, borders, backgrounds
- **Success**: Emerald-500 â€” won deals, completed tasks
- **Warning**: Amber-500 â€” approaching deadlines, attention needed
- **Danger**: Rose-500 â€” lost deals, delete actions, errors
- **Background**: White (`#FFFFFF`) content, Slate-50 (`#F8FAFC`) page background

### Typography Scale
| Role        | Size  | Weight   | Usage                        |
|-------------|-------|----------|------------------------------|
| Display     | 30px  | Semibold | Page titles                  |
| Heading     | 20px  | Semibold | Section headers, card titles |
| Subheading  | 16px  | Medium   | Table headers, tab labels    |
| Body        | 14px  | Regular  | Default text, table cells    |
| Caption     | 12px  | Regular  | Timestamps, helper text      |

## 8. Accessibility Guidelines

- All interactive elements are keyboard-accessible with visible focus indicators (2px indigo-500 outline)
- Color is never the sole indicator of meaning (icons + text accompany color states)
- Minimum contrast ratio: 4.5:1 for body text, 3:1 for large text
- All images and icons have appropriate `alt` text or `aria-label`
- Form inputs have associated `<label>` elements
- Modals trap focus and return focus to trigger on close
- Toast notifications include `role="alert"` for screen reader announcement
- Drag-and-drop (Kanban) has keyboard alternative: stage dropdown selector in deal card/detail
- Skip navigation link for keyboard users
- Responsive text scaling supports up to 200% browser zoom
